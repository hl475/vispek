## how to run run_file_io.py
You need to specify in_path and out_path, e.g.,
'''
python3 -m vispeck.examples.run_file_io \
        --inpath /Users/huaminli/Downloads/data \
        --outpath /Users/huaminli/Desktop/vispek/data

'''
