## how to run run_raw_file_io.py
You need to specify in_path and out_path, e.g.,
```
python3 -m vispek.examples.run_raw_file_io --in_path /Users/huaminli/Downloads/data --out_path /Users/huaminli/Desktop/vispek/data
```
